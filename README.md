## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/miranas11/ALBA.git
    ```

2. Install the dependencies and start servers:
    ```bash
    cd backend
    npm install
    node server.js
    ```
    ```bash
    cd frontend
    npm install
    npm start
    ```
